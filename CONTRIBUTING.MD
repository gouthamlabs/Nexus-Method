# Contributing to NEXUS Method

Thank you for your interest in NEXUS. This is a methodology project — contributions are welcome and valued.

## 🎯 What Makes a Valuable Contribution

In rough order of impact:

1. **New worked examples in different domains** (geopolitics, education, climate policy, defense, agriculture, energy)
2. **Domain pack extensions** (specialized tuning for regulated industries, specific markets, or technical fields)
3. **Honest critiques** of where the method fails — the most valuable contributions of all
4. **Mental model additions** to the polymath library, with proper attribution
5. **Translations** of prompts into other languages
6. **Documentation improvements** with concrete examples

## ⚠️ What We Don't Want

- Generic "this is great" PRs
- Adding more lenses just to add more — Polymath Mode is intentionally bounded
- Removing the honest caveats — NEXUS is honest about its limits and will remain so
- Politically motivated edits to the Master Directive

## 🧪 How to Contribute a Worked Example

The bar is high. A worked example must demonstrate:

1. A genuinely hard problem (where single-domain thinking has demonstrably failed)
2. Real cross-lens convergence (multiple lenses pointing to the same diagnosis)
3. At least one operationally distinct insight from mainstream frameworks
4. Calibrated confidence levels for every claim
5. **Brutal self-critique** — what worked, what didn't, where the method was weak

Use the existing examples in `examples/` as the template.

## 🎭 How to Add a Domain Pack

Domain packs in `prompts/domain_packs/` should:

1. Tune archetypes to domain-specific roles
2. Add domain-specific data sources or evidence standards
3. Calibrate confidence levels appropriate to the domain
4. Provide one short worked example showing the pack in use

## 📝 Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Include rationale in the PR description
5. Wait for review

## 🤝 Code of Conduct

See `CODE_OF_CONDUCT.md`. Disagree freely; treat people decently.
